## üõ†Ô∏è Getting Started

**Prerequisites:**  
- Node.js

### Local Installation

1. Clone this repo & enter project directory
   ```bash
   git clone https://github.com/medoxisto/Ai-career---Education.git
   cd Ai-career---Education
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set your Gemini API key:
   - Copy `.env.local.example` to `.env.local` and set `GEMINI_API_KEY`
4. Run the app:
   ```bash
   npm run dev
   ```
5. Visit `http://localhost:3000` to start your career journey!
